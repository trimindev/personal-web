## What is CSS and what is its relationship with HTML5 ?

- <span>•</span>CSS, hoặc Cascading Style Sheet, là style sheet language được sử
  dụng với HTML5 để định dạng và hiển thị các elements.

- <span>•</span>Trong khi HTML5 xác định structure của một page, CSS chỉ định
  kiểu của document, bao gồm page layout, colors và fonts.

## What is the Box model ? Which CSS properties are a part of it ?

- <span>•</span>Là một cái box bao bọc xung quanh HTML element.

- <span>•</span>Nó bao gồm : margin, padding, border, content.

## What are the advantages of using CSS ?

- <span>•</span>responsive, easy to maintain, bandwidth.

## What is the difference between inline, inline-block, and block ?

- <span>•</span>Block Element luôn bắt đầu ở new line và chiếm hết width của
  parent element hoặc tự set width.

- <span>•</span>Inline Element bắt đầu ở same line của content hoặc tag bên cạnh
  chúng.

- <span>•</span>Inline Block Elements giống block element nhưng bắt đầu ở same
  line của content hoặc tag bên cạnh chúng.

## What are Pseudo elements and Pseudo classes ?

- <span>•</span>Pseudo-elements cho phép chúng ta tạo items không exist trong
  document tree.
- <span>•</span>Ex: ::after, ::before, ::first-letter, :: first-line,
  ::selection.

- <span>•</span>Pseudo-classes chọn các element thông thường nhưng trong các
  điều kiện nhất định.

- <span>•</span>Ex: :link, :visited, :hover, :active, :focus.

## How is border-box different from content-box ?

- <span>•</span>content-box là box-sizing property mặc định,nó set width và
  height của element chỉ bao gồm content.

- <span>•</span>border-box property set width và height của element bao gồm cả
  border, padding và content.

## What is float property ?

- <span>•</span>được sử dụng để định vị element sang left hoặc right của
  container.

## What is a z-index, how does it function ?

- <span>•</span>Xác định order của element được hiển thị như thế nào khi
  overlapping.

- <span>•</span>Values of z-index : auto, number, initial, inherit.

## What are the properties of flexbox ?

- <span>•</span>flex-direction, flex-wrap, flex-flow, justify-content,
  align-items, align-content.

## What do the following CSS selectors mean ?

- <b>div, p | div p | div ~ p | div + p | div > p</b>

- <span>•</span>div, p select all div and p elements.
- <span>•</span>div p select all p elements that inside div elementss.
- <span>•</span>div ~ p select all p elementss that have p elements preceed.
- <span>•</span>div + p select all p elements placed immediately after the div
  elements.
- <span>•</span>div > p select all p elements which has div as an immediate
  parent.
