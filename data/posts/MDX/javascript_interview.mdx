## What is JavaScript ?

- <span>•</span>
  Là web scripting language phổ biến nhất, sử dụng cho cả client-side và server-side,
  có hỗ trợ OOP.

- <span>•</span>
  JavaScript code có thể được chèn vào HTML và được thực thi bởi web browser.

- <span>•</span>
