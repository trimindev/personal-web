## What is HTML ?

- <span>•</span>Là ngôn ngữ đánh dấu siêu văn bản (Hyper Text Markup Language).

- <span>•</span>Được sử dụng để cấu trúc và hiển thị content của 1 website.

- <span>•</span>File HTML sẽ bao gồm nhiều element giúp cho browser hiển thị
  content trở nên tương tác và năng động hơn.

## What is DOM ?

- <span>•</span>Là một programming interface đại diện cho các document HTML dưới
  dạng các nodes và objects

- <span>•</span>Giúp programs có thể dễ dàng thay đổi structure, style và
  content của document

## What are the key differences between HTML and HTML5?

- <span>•</span> HTML5 hỗ trợ video, graphics, và audio, whereas HTML chỉ hỗ trợ
  chúng thông qua third-party extensions.

- <span>•</span>HTML5 thân thiện với mobile, whereas HTML thì không.

- <span>•</span>HTML5 tương thích với tất cả các web browsers chính, whereas
  HTML thì không

- <span>•</span>HTML5 cung cấp một số options cho local storage, whereas HTML
  chỉ cung cấp cookies

- <span>•</span>HTML5 hỗ trợ multi-threading, whereas HTML chỉ hoạt động trong 1
  thread

## What are elements ?

- <span>•</span>Là 1 phần của HTML code, có nhiệm vụ hướng dẫn browser cấu trúc
  và hiển thị HTML document.

- <span>•</span>1 Element thông thường sẽ bao gồm opening tag, closing tag,
  content nằm ở giữa.

## What are attributes ?

- <span>•</span>Cung cấp information bổ sung cho element.

- <span>•</span>Được đặt sau tên tag, và chỉ xuất hiện ở opening tag.

## What are meta tags ?

- <span>•</span>Được đặt trong phần head để cung cấp information về page giúp
  tối ưu hóa công cụ tìm kiếm.

- <span>•</span>Ex: title, description, content-type, keyword,...

```html
<head>
  <meta name="description" content="description" />
  <title>Website</title>
</head>
```

## What is iframe ?

- <span>•</span>Là tag giúp embed html,image, video, web khác vào website của
  mình.

```html
<iframe src="https://abc.com" height="300px" width="300px"></iframe>
```

## What is the DOCTYPE Declaration ?

- <span>•</span>mục đính để browser nhận biết file HTML đó được viết ở version
  nào.

## How many way to specify color in HTML ?

- <span>•</span>Có 4 cách: name, RGB / RGBA , HEX, HSL / HSLA.

## What is differences between `<div>` and `<span>` tag ?

- <span>•</span>div là block element, span là inline element.

- <span>•</span>Nên dùng div để bọc các phần của web page và span để bọc chứa
  các phần nhỏ của element.

- <span>•</span>div chấp nhận align attribute còn span thì không.
